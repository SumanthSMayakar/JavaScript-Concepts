What is a Map

A map is a data structure that can be used to store values mapped to a key

Just like an object, the data in a Map is stored as a key-value pair

Mao is an iterable

The difference between an object and a map is that, the key in a Map can be a value of any datatype:

The keys or properties of an object are always a string

The keys in a map can be value of any data type. They can even be an object, an array or another map
