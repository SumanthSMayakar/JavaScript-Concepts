What is Destructuring

Destructuring is a special assignment syntax in javascript that allows us to unpack arrays or objects into a bunch of variables

There are two types of destructuring

array destructuring

object destructuring

Object Destructuring

The variable name and the property name should be same while destructuring the Object

The object destructuring is based on the reference of the object

The reference name can be changed by giving alias name with reference of the object name (Eg: let {name: firstName, age: myAge, gender}) using colon
