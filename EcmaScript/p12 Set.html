What are Sets in JAVASCRIPT

A set is just a collection of unique values

sets can never have duplicate entries

sets cannot be accessed by index number

set cannot be retrived

set can be iterable

A sei is different from array in the way that

Elements of a set are unique

The order of element in the set is irrelevant

Properties & Methods of Set

size : returns the numbers of elements in a set

has() : returns true if a certain elements is present in the set

add() : adds a new unique elements to the set

delete() : Delete an elements from the set

clear() : Delete all the elements of the set
